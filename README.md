# MSI-Z390M-Mortar
opencore efi for MSI Z390M Mortar

基于OC 0.6 正式版编写的适用于微星msi Z390M mortar主板的efi引导文件

使用前请记得修改三码

bios版本：7C00V18（当前最新版）
cpu: 9900es qqz5
主板：微星z390m mortar
内存：cuso ddr4 2666 16g*2 + juhor ddr4 2666 8g*2
显卡：华硕dual rx580 o8g
硬盘：intel 660p 512g
有线网卡：主板自带的有线intel i219v
免驱无线卡：bcm94360cd
